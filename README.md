### VuePressスタートアップ
```bash
# yarnの初期化
$ yarn init

# VuePressのインストール
$ yarn add -D vuepress

# README.mdファイルの作成
$ echo "Hello Vuepress" > README.md

# ローカルサーバー起動
$ yarn vuepress dev
```
